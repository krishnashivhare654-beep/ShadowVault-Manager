System Architecture
🛡️ Cryptographic Engine

ShadowVault uses AES-256 Symmetric Encryption via Python's cryptography.fernet. This ensures that every entry is cryptographically signed and encrypted before being written to the database.

🔑 Zero-Knowledge Policy

The system is designed so that data can only be decrypted using the unique master.key file generated locally. This ensures total privacy.
