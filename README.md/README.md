# 🛡️ ShadowVault Password Manager

A secure, local-first password manager built with **Python**, **Flask**, and **AES-256 Encryption**.

## 🚀 Features
* **Zero-Knowledge Encryption:** Passwords are encrypted before being stored.
* **Master Key Protection:** Uses a local key file for symmetric encryption.
* **Clean UI:** Simple web dashboard for managing credentials.
* **Database:** SQLite backend for lightweight portability.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Framework:** Flask
* **Security:** Cryptography.fernet
* **Database:** SQLAlchemy
